Step 1: Create the Base Skin
1. Create a new site skin.
2. Copy and paste the Base Code into the CSS box.
3. Scroll down to the "Advanced" section and check the box for "Parent Only".
4. Click "Submit" to save.

Step 2: Create the Color Theme Skin
1. Create another new site skin (choose either *Cream Cocoa* or *Grand Budapest*).
2. Copy and paste the Color Code of your choice into the CSS box.
3. Scroll down to the "Advanced" section again.
4. Find "Parent Skins" and click the "Add parent skin" button.
5. In the input field that appears, search for the name of the Base Skin you just created in Step 1, and select it.
6. Click "Submit" to save and apply.
